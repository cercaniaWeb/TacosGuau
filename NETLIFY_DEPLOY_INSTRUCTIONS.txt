INSTRUCCIONES DE DESPLIEGUE EN NETLIFY

Este archivo contiene los pasos para desplegar tu proyecto en Netlify. Es temporal y puedes eliminarlo una vez que hayas completado el despliegue.

1.  **Asegúrate de que tu código esté en un repositorio de Git:**
    *   Si aún no lo has hecho, inicializa un repositorio de Git en la raíz de tu proyecto:
        `git init`
    *   Añade todos tus archivos:
        `git add .`
    *   Haz tu primer commit:
        `git commit -m "Initial commit"`
    *   Crea un repositorio en tu proveedor de Git (GitHub, GitLab, Bitbucket) y sube tu código a ese repositorio. Por ejemplo, para GitHub:
        `git remote add origin <URL_DE_TU_REPOSITORIO>`
        `git push -u origin main` (o master, según tu rama principal)

2.  **Conecta Netlify con tu Repositorio:**
    *   Ve a https://app.netlify.com/ e inicia sesión (o crea una cuenta).
    *   Haz clic en **"Add new site"** o **"New site from Git"**.
    *   Selecciona tu proveedor de Git (GitHub, GitLab, Bitbucket).
    *   Autoriza a Netlify para acceder a tus repositorios.
    *   Elige el repositorio donde tienes el código de tu proyecto `tacos-guau-app`.

3.  **Configura las Opciones de Despliegue:**
    Netlify intentará detectar la configuración automáticamente, pero es bueno confirmarla:
    *   **Owner:** Tu cuenta de Git.
    *   **Repository:** El nombre de tu repositorio.
    *   **Branch to deploy:** `main` (o la rama que uses para el despliegue, como `master`).
    *   **Build command:** `npm run build`
    *   **Publish directory:** `build`
    *   **Functions directory:** `netlify/functions`

4.  **Configura las Variables de Entorno (¡MUY IMPORTANTE!):**
    *   **Antes de hacer el primer despliegue**, ve a la configuración de tu sitio en Netlify (después de haber seleccionado el repositorio, pero antes de hacer clic en "Deploy site").
    *   Navega a **Site settings > Build & deploy > Environment variables**.
    *   Añade las siguientes variables:
        *   `GMAIL_USER`: `tacosguaupanda@gmail.com`
        *   `GMAIL_PASS`: `fljv pohb oijg krro` (tu contraseña de aplicación de Google)
        *   `NETLIFY_SITE_URL`: La URL que Netlify le asignará a tu sitio (por ejemplo, `https://tu-nombre-de-sitio.netlify.app`). Puedes dejar esta en blanco para el primer despliegue y Netlify la rellenará automáticamente, o puedes poner un valor de marcador de posición como `https://example.com` y luego actualizarlo con la URL real de tu sitio Netlify una vez que se despliegue.

5.  **Desplegar el Sitio:**
    *   Una vez que hayas configurado todo, haz clic en **"Deploy site"**.
    *   Netlify clonará tu repositorio, ejecutará el comando de construcción, desplegará los archivos de la carpeta `build` y configurará tus funciones de Netlify.

¡Tu proyecto estará en línea! Cada vez que hagas un `git push` a la rama que configuraste, Netlify automáticamente construirá y desplegará los cambios.
